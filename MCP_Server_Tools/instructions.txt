### MainCode/ (To Run and Test Code Individually)
This directory contains the original code used during the course tutorial. It includes:
- `Scenario1/`: Basic MCP calculator implementation (run: npx @modelcontextprotocol/inspector python path/to/scenario1/file)
- `Scenario2/`: FastAPI integration with MCP (run: python path\to\scenario2\file and npx @modelcontextprotocol/inspector https://localhost:8002/mcp)
- `Scenario3/`: FreeCodeCamp RSS feed reader (run: npx @modelcontextprotocol/inspector python path/to/scenario3/file)
- `deployment/`: Deployment configuration examples

### Explanatory Version/ is the directory of explanation of code in MainCode

## Getting Started with MCP

To use MCP in Visual Studio Code, you need to configure it properly:

0. Install dependencies after creating virtual env using pip/conda/uv and also Download Github Copilot Extension on VSCode 
and run:npm install -g @modelcontextprotocol/inspector to inspect

1. Create a `.vscode` folder in your project root:
```bash
mkdir .vscode
```

2. Create an `mcp.json` file inside the `.vscode` folder with this template using "sample_mcp.json" provided in the folder

3. Then open Github Copilot and select model and tools, then it will be prompted to open and run mcp server, allow it to run and test
